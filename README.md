# 100daysofcode twitter bot

A simple Twitter bot which will tweets "#100daysofcode" tweet for you if you forget to do so !

## Installation

- `npm i`
- Add a file `.env` in root directory with key values as,

```
CONSUMER_KEY=
CONSUMER_SECRET=
ACCESS_TOKEN=
ACCESS_TOKEN_SECRET=
USER_NAME=
USER_ID=
APP_NAME=
GITHUB_URL=
PORT=
```

- Run app `npm start`.

### That's it! Happy coding!
